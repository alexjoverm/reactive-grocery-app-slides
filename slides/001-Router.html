Usar Angular Router

En este sprint crearemos los componentes necesarios para añadir componentes a una lista. Lo haremos en una nueva página o container GroceriesDetail

Para ello utilizaremos Angular Router, localizado en el módulo @angular/router. Si has utilizado el starter de SystemJS o Angular CLI, ya está incluido.

Angular 2 Router es muy versátil y poténte. Aquí veremos lo básico.
